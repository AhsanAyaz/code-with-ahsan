### Event Details
- **Date:** April 11, 2026
- **Time:** 1:00 PM PKT
- **Speaker:** [Muhammad Ahsan Ayaz](https://www.linkedin.com/in/ahsanayaz/)
- **Venue:** Online via Google Meet
- **Price:** Free (Registration Required)

## What to Expect

- **Terminal-First Workflow with Gemini CLI:** Learn how to configure and use the Gemini command-line interface to help you write, review, and debug your code seamlessly.
- **Fundamentals of Google's ADK:** Get a clear introduction to Google's Agent Development Kit. Learn about setting up agent states, tracking, memory, and executing workflows.
- **Integrating Real-World APIs via MCP:** Discover how to connect your AI travel agent to real-world services (like Airbnb) using the Model Context Protocol (MCP).
- **Live Search & Grounding:** Equip your agent with search capabilities to query real-time data, like current weather forecasts or live event dates.
- **Visual Creativity with NanoBanana:** See how to hook up image generation capabilities to build custom room and destination previews on the fly.
- **Live Q&A:** Bring your trickiest AI integration questions and get answers in real time!

## Who Should Attend
- **Web Developers & Software Engineers** who want to advance from simple "prompt engineering" to building stateful, production-ready AI agent architectures.
- **AI Enthusiasts & Tinkerers** eager to experience Google's developer toolkit and understand the concept of "vibecoding".
- **Command Line Lovers** who want to superpower their shell environment with Gemini CLI.
- **Anyone curious about Multi-Agent systems** and how to orchestrate collaboration between specialized AI helpers.
