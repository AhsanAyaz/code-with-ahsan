`sub_agents` vs `AgentTool` distinction (RESEARCH §3.5 lines 539-546): `sub_agents=[specialist]` = LLM-driven transfer (control moves to specialist); `tools=[AgentTool(agent=specialist)]` = synchronous call-and-return (specialist invoked, result inlined, original agent stays in control). def search_blog_posts(query: str) -> dict:

    Returns:

    q = (query or "").strip()

    result = fetch_blog_posts(q)
    if not result["ok"]:
        return {"status": "error", "message": result["error"], "posts": []}

    all_posts = (result["data"] or {}).get("posts", [])
    posts = [_shape_blog_post(p) for p in all_posts[:_BLOG_LIMIT]]

    return {

        "query": q,
