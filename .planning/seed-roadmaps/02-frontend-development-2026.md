# Frontend Development 2026: Resumability & AI-Generated UIs

## Framework Decision Tree

### When to use what?

**Next.js (React)**
- Full-stack applications
- SEO-critical sites
- E-commerce, SaaS
- **Default choice for most projects**

**Astro**
- Content-focused sites (blogs, docs)
- Marketing websites
- When you need minimal JS

**Qwik**
- Performance-critical apps
- Mobile-first experiences
- When resumability matters

**Vite + React**
- SPAs without SSR
- Internal tools
- Prototypes
