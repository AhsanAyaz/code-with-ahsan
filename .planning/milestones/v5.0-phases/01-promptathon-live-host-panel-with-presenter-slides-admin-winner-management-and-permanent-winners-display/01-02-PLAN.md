**HostPanel.tsx — section state machine + keyboard handler**

Sections in order (index 0–9):
```
0: keynote, 1: community, 2: sponsors, 3: judges,
4: mentors, 5: rollcall, 6: themes, 7: twist, 8: sendoff, 9: winners
```

State:
- `sectionIndex: number` (0–9)
- `revealedCount: number` — resets to 0 on section change; used by roll call (teams), judges (cards), winners (podium steps)
- `controlsVisible: boolean` (default true)
- `twistPhase: "idle" | "countdown" | "revealed"` — resets to "idle" on section leave

Keyboard handler in useEffect (cleanup on unmount):
- `N` or `ArrowRight` when no sub-item to advance: call `advanceSection()`
- `P`: call `retreatSection()`
- `H`: toggle controlsVisible
- `F`: `document.documentElement.requestFullscreen?.()` (guard: inside useEffect so document is defined)
- `Space` or `ArrowRight`: call `handleAdvance()` — within-section advance logic:
  - Section 5 (rollcall): if revealedCount < 10, increment revealedCount; else advanceSection()
  - Section 3 (judges): if revealedCount < 4, increment revealedCount; else advanceSection()
  - Section 7 (twist): if twistPhase === "idle" → set "countdown"; else if "revealed" → advanceSection()
  - Section 9 (winners): if revealedCount < 3, increment revealedCount (3rd→2nd→1st); else advanceSection()
  - All other sections: advanceSection()

Note: N and Space/ArrowRight both call handleAdvance in the keyboard handler — N is section-level (always next section), Space/→ is within-section-aware.

Actually implement as: N always calls advanceSection(), Space/→ calls handleAdvance() (section-aware).

Render: `AnimatePresence mode="wait"` wrapping a `motion.div` with `key={sectionIndex}`, transitions as per research patterns. Only render the active section component.

Pass all needed props to each section:
- rollcall: `revealedCount`, `onReveal: () => setRevealedCount(c => c + 1)`
- judges: `revealedCount`
- twist: `twistPhase`, `onStartCountdown: () => setTwistPhase("countdown")`, `onRevealed: () => setTwistPhase("revealed")`
- winners: `revealedCount`, `onReveal: () => setRevealedCount(c => Math.min(c + 1, 3))`

Section indicator: always rendered on top of everything, even when controls hidden:
