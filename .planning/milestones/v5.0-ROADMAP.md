# Milestone v5.0: CWA Promptathon 2026

**Status:** ✅ SHIPPED 2026-04-21
**Phases:** 1 (single phase)
**Total Plans:** 3

## Overview

Event-specific milestone delivering the live-stream presenter panel, admin winner management, and permanent winners podium for the CWA Prompt-A-Thon 2026 hackathon event. Runs in a dedicated subtree under `/events/cwa-promptathon/2026/` (host route, public page components) and `/admin/events/` (winner management), with a shared Firestore winners API backing both.

## Phases

**Goal:** Fullscreen live-stream presenter panel for the CWA Prompt-A-Thon 2026 host, with admin winner management and a permanent podium display on the public event page.
**Depends on:** Phase 0 (prior milestones — v4.0 admin infrastructure)
**Plans:** 3 plans

Host auth decoupled from Firebase user by reusing only `ADMIN_TOKEN_KEY`. Post-execution polish commits covered font consistency, sponsor/judge updates, and mentors/organizers + wrap-up slide.

## Milestone Summary

**Key Decisions:**

- Public GET on winners API (no credentials required) to let the public display panel read without auth complexity
- `HostAuthGate` imports only `ADMIN_TOKEN_KEY` from `AdminAuthGate` — token-only auth with no Firebase user dependency
- `TwistRevealSection` owns its countdown internally via `setInterval` in `useRef`; parent `HostPanel` only flips `twistPhase` state
- `WinnersSection` uses `prevRevealedCount` ref to detect 2→3 transition for confetti (prevents double-fire)
- Admin form uses `PLACEMENTS` config array to DRY up 3 placement sections; `WinnersDisplay` returns null until `announcedAt` confirmed

**Issues Resolved:**

- Judge list corrections (Muhammad Ali added then removed; Imran Siddiqui replaced by Muhammad Ali temporarily)
- Sponsor placeholder copy cleaned up before event
- Winners reveal confetti double-fire prevented via ref-based transition detection
- Admin form "save before all teams selected" bug — now requires all 3 teams

**Issues Deferred:**

- Multi-year event support: host route hardcoded to `/events/cwa-promptathon/2026/`; future years will need route templating
- Winners API not generalized — tied to `cwa-promptathon-2026` event ID; other events would need their own doc structure

**Technical Debt Incurred:**

- Admin Events page is minimal (list view hard-coded to the single CWA Promptathon event); will need generalization for future event types
