## Security Finding

**CRITICAL: `GET /api/roadmaps?admin=true` has no auth check**

File: `src/app/api/roadmaps/route.ts` line 230

The `admin=true` query parameter branch executes without authentication. Any anonymous user can access pending roadmaps, draft metadata, and unpublished content intended only for admins. All mutating endpoints (POST/PUT/DELETE) correctly verify auth — only the GET list path is exposed.
