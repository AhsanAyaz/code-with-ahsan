# Hello World Instructions

When this skill is activated by the phrase "hello world":

1. **System Check**: Execute the bundled Node.js script `scripts/sys-info.js` to get the current environment details.
2. **Greeting**: Display a large ASCII art "Hello World". Use the script `script/generate-ascii.js` to generate the art to generate the art.
3. **Report**: Summarize the data from the script in a clean table format.
